## 📖 Description

**CleanInstafy** est une application desktop moderne qui synchronise automatiquement votre lecture musicale Windows vers vos **Instagram Notes**. 

Peu importe l'application que vous utilisez (Spotify, YouTube Music, Apple Music, VLC, etc.), CleanInstafy détecte automatiquement ce que vous écoutez et le partage sur votre profil Instagram en temps réel !

### 🎯 Pourquoi CleanInstafy ?

- 🎵 **Universel** : Fonctionne avec toutes les applications de musique Windows
- 🚫 **Sans API Spotify** : Pas besoin de clé API ou configuration complexe
- 🔒 **Sécurisé** : Vos identifiants restent sur votre machine, aucune donnée n'est envoyée à des serveurs tiers
- 🎨 **Interface moderne** : Design élégant qui s'adapte aux couleurs de votre musique
- ⚡ **Léger et rapide** : Moins de 100MB, démarre en moins de 3 secondes

## 🚀 Installation

### Méthode Simple (Recommandée)

1. **Téléchargez** `CleanInstafy.exe` depuis la [page des releases](https://github.com/ErrorNoName/CleanInstafy/releases/download/CleanInstafy/CleanInstafy.rar)

2. **Double-cliquez** sur l'exécutable pour lancer l'application

3. **Connectez-vous** avec vos identifiants Instagram

4. **Activez** la synchronisation avec le bouton toggle

5. **Profitez** ! Votre musique apparaît automatiquement sur Instagram 🎉

### ⚠️ Note de Sécurité Windows

Si Windows Defender affiche un avertissement "Application non reconnue" :

1. Cliquez sur **"Informations complémentaires"**
2. Puis sur **"Exécuter quand même"**

> Ceci est normal pour les applications non signées numériquement. CleanInstafy est 100% sûr et open-source.

## 📱 Utilisation

<div align="center">

### Interface Principale

### Étape 1️⃣ : Connexion Instagram

1. Entrez votre **nom d'utilisateur** Instagram
2. Entrez votre **mot de passe**
3. Si vous avez activé la 2FA, entrez le **code de vérification**
4. Cliquez sur **"Se connecter"**

> 🔒 **Vos identifiants sont stockés de manière sécurisée** et ne quittent jamais votre ordinateur.

### Étape 2️⃣ : Activer la Synchronisation

Une fois connecté, vous verrez :
- **📊 Statistiques** : Nombre de synchronisations effectuées
- **🎵 Lecture en cours** : Informations sur la musique actuelle
- **🔄 Toggle** : Activez/désactivez la synchronisation

Basculez simplement le bouton **"Activer"** pour commencer !

### Étape 3️⃣ : Lancez votre musique préférée

Ouvrez **n'importe quelle application musicale** sur Windows et commencez à écouter. CleanInstafy fera le reste ! 🎧

## ❓ FAQ et Dépannage

<details>
<summary><b>🔴 L'application ne détecte pas ma musique</b></summary>

**Solutions :**

1. **Vérifiez que votre application musicale est en lecture active**
   - Mettez en pause puis relancez la musique
   - Essayez de changer de piste

2. **Redémarrez l'application musicale**
   - Fermez complètement Spotify/YouTube Music
   - Relancez l'application
   - Attendez quelques secondes

3. **Redémarrez CleanInstafy**
   - Fermez CleanInstafy
   - Relancez l'application
   - Réactivez la synchronisation

4. **Vérifiez les permissions Windows**
   - Ouvrez les **Paramètres Windows**
   - Allez dans **Confidentialité > Informations d'application**
   - Assurez-vous que les applications peuvent accéder aux informations de compte

</details>

<details>
<summary><b>🔑 Erreur de connexion Instagram</b></summary>

**Causes courantes :**

1. **Identifiants incorrects**
   - Vérifiez votre nom d'utilisateur (sans @)
   - Vérifiez votre mot de passe
   - Essayez de vous connecter sur instagram.com pour confirmer

2. **Authentification à deux facteurs (2FA)**
   - Si vous avez activé la 2FA, entrez le code de vérification
   - Le code change toutes les 30 secondes, assurez-vous d'utiliser le code actuel
   - Si vous utilisez une application d'authentification, ouvrez-la pour obtenir le code

3. **Compte temporairement bloqué**
   - Instagram peut bloquer temporairement les connexions suspectes
   - Attendez 30 minutes à 1 heure
   - Connectez-vous manuellement sur instagram.com depuis un navigateur
   - Réessayez après validation

4. **Défi de sécurité Instagram**
   - Instagram peut demander une vérification supplémentaire
   - Connectez-vous sur instagram.com et complétez le défi
   - Réessayez dans CleanInstafy

**💡 Astuce :** Pour éviter les blocages, espacez vos tentatives de connexion d'au moins 5 minutes.

</details>

<details>
<summary><b>⚠️ "Session expirée" après quelques heures</b></summary>

**Solutions :**

1. **Reconnexion automatique**
   - CleanInstafy tente automatiquement de se reconnecter
   - Si cela échoue, reconnectez-vous manuellement

2. **Session Instagram révoquée**
   - Allez dans les paramètres Instagram > Sécurité > Connexions actives
   - Si vous voyez beaucoup de connexions, supprimez les anciennes
   - Reconnectez-vous dans CleanInstafy

3. **Changement de mot de passe**
   - Si vous avez changé votre mot de passe Instagram, vous devez vous reconnecter

</details>

<details>
<summary><b>🎵 La musique ne s'affiche pas sur Instagram</b></summary>

**Vérifications :**

1. **La synchronisation est-elle activée ?**
   - Vérifiez que le toggle est sur "ON" (bleu)

2. **Instagram Notes est-il disponible ?**
   - Les Notes Instagram ne sont pas disponibles dans tous les pays
   - Vérifiez sur votre application Instagram mobile

3. **Limite de caractères**
   - Les Notes Instagram sont limitées à 60 caractères
   - Les titres longs sont automatiquement tronqués

4. **Délai de propagation**
   - Il peut y avoir un délai de 5-10 secondes
   - Rafraîchissez votre application Instagram

</details>

<details>
<summary><b>🖥️ L'application ne démarre pas</b></summary>

**Solutions :**

1. **Vérifiez Windows Defender**
   - L'antivirus peut bloquer l'exécution
   - Ajoutez une exception pour CleanInstafy.exe

2. **Réinstallez Microsoft Visual C++ Redistributable**
   - Téléchargez depuis [Microsoft](https://aka.ms/vs/17/release/vc_redist.x64.exe)
   - Installez et redémarrez votre PC

3. **Exécutez en tant qu'administrateur**
   - Faites un clic droit sur CleanInstafy.exe
   - Sélectionnez "Exécuter en tant qu'administrateur"

4. **Vérifiez les journaux**
   - Consultez le fichier `cleaninstafy.log` dans le dossier de l'application
   - Partagez les erreurs dans les [Issues GitHub](https://github.com/ErrorNoName/CleanInstafy/issues)

</details>

<details>
<summary><b>🔄 L'application se fige ou ne répond plus</b></summary>

**Solutions :**

1. **Fermez et relancez**
   - Utilisez le gestionnaire des tâches (Ctrl + Shift + Esc)
   - Terminez le processus CleanInstafy
   - Relancez l'application

2. **Vérifiez les ressources système**
   - CleanInstafy utilise très peu de ressources
   - Si votre PC est surchargé, fermez d'autres applications

3. **Mettez à jour l'application**
   - Vérifiez qu'il n'y a pas de nouvelle version disponible
   - Téléchargez la dernière version depuis les releases

</details>
